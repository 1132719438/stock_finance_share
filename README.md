### 关于 HOLD LE

---

HOLD LE 是一个交易策略学习及工具网站，旨在传播正确的价值投资观念及技巧，学习如何通过合理运用各种分析方法，提高交易成功率，提升交易回报。

HOLD LE 适用于包括股票、区块链货币、外汇等在内的金融交易，是经过笔者多次交易实践检验的高效交易策略，学习完毕后，能够很大程度提高金融交易的判断和操作能力。

**核心内容包括 ：**
- 投资交易的正确观念

- 用 HOLD LE 工具分析搜索高概率增长的股票

- 筛选出具备中长期成长价值的投资标的

- 最佳交易时间的出现节点（最大化收益、最小化风险）

- 对交易中买入、卖出等各个环节的最优处理

- 高效分散投资的方法

- 避免交易过程中的各种陷阱

- 如何运用于比特币、外汇交易

- ……

### 网站开发技术栈

---

- [HOLD LE on GitHub](https://github.com/emilesu/stock_finance_share)

- Ruby on Rails （网站主程框架 MVC）

- bootsrap3     （基础前端框架）

- mysql         （数据库类型）

- devise        （Rails 用户管理挂件）

- kaminari      （Ruby 范围与搜索器）

- omniauth - omniauth-wechat-oauth2     （第三方 微信登录）

- redcarpet     （markdown 编译）

- chartkick     （图表处理 JS）

- qiniu         （七牛云存储）

- rest-client + nokogiri        （数据抓爬挂件）

- sidekiq       （异步处理）

- figaro        （密匙管理）
